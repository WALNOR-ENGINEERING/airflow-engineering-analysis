# Duct Bend Flow Behavior

![Cover](cover.png)

---

# Engineering problem

Changes in airflow direction often generate:

• flow separation  
• vortex formation  
• unstable airflow regions  
• additional pressure losses  

These effects reduce efficiency of airflow systems.

---

# Analysis overview

Airflow behavior inside a 90° duct bend
was investigated to observe
flow separation and vortex structures.

The analysis focuses on airflow instability
caused by sudden geometry changes.

---

# Key observations

• flow separation near the inner wall  
• secondary vortex structures  
• uneven velocity distribution  
• localized pressure losses  

---

# Engineering relevance

Understanding airflow behavior inside bends helps:

• reduce pressure losses  
• stabilize airflow  
• improve duct geometry  
• increase system efficiency
