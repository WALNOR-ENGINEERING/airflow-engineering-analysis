# CFD Airflow Simulation

Engineering CFD studies focused on airflow behavior,
turbulence and pressure losses in technical systems.

This repository contains several simulation examples
demonstrating airflow analysis in ducts, pipes and
separation systems.

Topics covered:

• turbulent pipe flow
• vortex formation
• duct flow behavior
• cyclone separation
• pressure drop analysis

Tools used:

• OpenFOAM
• FreeCAD CFD
• ParaView

The purpose of these studies is to demonstrate
engineering simulation workflows and airflow analysis
methods used in industrial applications.
